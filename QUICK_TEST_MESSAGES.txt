# Quick Test Messages - Copy & Paste

## Setup: Start a game first
"let's play a game"

## Category 1: Clear High-Confidence Intents
"end this game"
"I resign"
"reset board"
"end walkthrough"

## Category 2: Ambiguous (Need Clarification)
"end"
"stop"
"end this"
"quit"

## Category 3: Low Confidence / Fallback
"end something"
"do something"
"ending"

## Category 4: Confirmation Responses
"yes"
"y"
"no"
"n"

## Category 5: Clarifying Question Responses
"1"
"2"
"3"
"end the game"
"end walkthrough"

## Category 6: Context Tests
"end" (with only game active)
"end" (with game + walkthrough active)
"end" (with nothing active)

"edn game"
"please end this game now"
"don't end the game"
"how do I end the game?"
